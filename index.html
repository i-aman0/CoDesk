HTML

HTML is a markup language that is primarily used to design the basic structure of a web page. HTML provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes, and other items. The default characteristics of every HTML element item are defined in the browser.

CSS

CSS stands for Cascading Style Sheets and is a style sheet language used for describing the presentation of a document written in HTML or XML. The name Cascading comes from the specified priority scheme to determine which style rule applies if more than one rule matches a particular element. The CSS specifications are maintained by W3C.

JavaScript

JavaScript often abbreviated as JS, is a programming language that is one of the core technologies of the World Wide Web. alongside HTML and CSS. It has dynamic typing, prototype based object-orientation and first class function. It is multi-paradigm. It has APIs for working with texts, dates, regular experessions, standard data structures and DOM.
