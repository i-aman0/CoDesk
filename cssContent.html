CSS

CSS stands for Cascading Style Sheets.

CSS is used to style web pages. It describes how an HTML document should look.

CSS Syntax

body {
      background-color: lightblue;
     }

h1 {
     color: white;
     text-align: center;
   }

CSS Syntax

The CSS Syntax consists of selectors and declaration rules.

Selectors consist of HTML Elements and declaration rules include CSS properties and their corresponding values.

CSS Margin

Margins are used to create space around elements, outside of any defined borders.

With CSS you have full control over the margins. These are properties for setting margin for each side of an element (top, right, bottom, left)

CSS Position

The position property specifies the type of positioning method used for an element.

The positions can be static, relative, fixed, absolute, and sticky.

CSS Opacity

The opacity property specifies the opacity or transparency of an element.

The opacity property can take value from 0.0 to 1.0. The lower the value the more the transparency.
